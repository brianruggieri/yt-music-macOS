Global names (CORRECTED — verified at RUNTIME in WKWebView, Spike B)
-------------------------------------------------------------------
IMPORTANT: both UMD bundles are built WITHOUT libraryExport:'default', so each
window global is the webpack module *namespace* — the real API lives under
`.default`. Always resolve as `window.<name>.default || window.<name>`.

window.butterchurn.default   (NOT window.butterchurn directly)
  createVisualizer(audioContext, canvas, {width, height, pixelRatio}) -> Visualizer
  isSupported()
  Visualizer#connectAudio(AudioNode)    -- connect an AudioNode as the audio input
  Visualizer#loadPreset(preset, blendTime)
  Visualizer#render()
  Visualizer#setRendererSize(width, height)

window.butterchurnPresets.default   (resolve the same way)
  getPresets() -> { [name: string]: preset }

Node-graph recipe (proven by Spike B harness in feed-spike.js)
--------------------------------------------------------------
AudioContext
  -> AudioWorkletNode (sine-sweep, 2ch out)
       |- viz.connectAudio(workletNode)   -- drives Butterchurn
       |- AnalyserNode                    -- for level readout in HUD
       |- GainNode(gain=0) -> destination -- keeps graph pulled (required)
